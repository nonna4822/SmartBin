# SmartBin
## โปรเจคนี้เป็นส่วนหนึ่งของรายวิชา Embedded System 
    เกี่ยวกับการทำ Smart device  โดยใช้ Atmega328p 
## อุปกรณ์ที่ใช้
* atmega328p
* PIR sensor
* Motor x 2
* Ultrasonic sensor
